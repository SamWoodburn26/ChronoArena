# ChronoArena (Project 2) - UI + Controls (Java Starter)

This is a minimal Java/Swing scaffold so you can start building the required **UI and controls** for ChronoArena.

## Run (local UI demo)

```bash
javac src/*.java
java -cp src ChronoArenaClientUI
```

## Keyboard controls

- Move: `W/A/S/D` or Arrow Keys
- Quit: `Esc` or `Q`

## Properties

The UI demo reads optional settings from `properties.properties` in the working directory.
If the file is missing, defaults are used.
